# うちの子 体重手帳

スマホが苦手な高齢の飼い主でも使いやすい、ペット用の体重管理ツールです。

## 主な機能

- 今日の体重、ごはん、うんち、散歩、薬、病院、メモを大きい入力欄で記録
- 体重推移グラフ、前回差、7日平均、急な増減の注意表示
- 最近の記録一覧
- 家族や病院に渡せるJSON出力、読み込み
- 印刷用表示
- スマホとパソコン対応

## 使い方

`index.html` をブラウザで開くだけで動きます。記録はブラウザ内に保存されます。
